# Reader UI Polish P0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the P0 reader UI polish so audio controls use a unified icon language, sentence play marks are removed, and missing audio gives honest feedback instead of simulated playback.

**Architecture:** Keep the existing reader page structure. Update the reader page only for UI state and event orchestration, and keep audio playback behavior in `src/utils/tts.ts`.

**Tech Stack:** Vue 3, uni-app, mp-weixin, TypeScript, Node UI smoke check.
